Safe-VerifiedAI

The Goal

To develop a Reinforcement Learning framework that mitigates bias and enforces fairness constraints in AI agents using a real-time Normative Fairness Shield, preventing discrimination against protected groups in automated decision-making systems.

Work Done

  • Synthetic Biased Environment: Built a custom CreditDataGenerator simulating a credit scoring environment where protected groups have artificially lowered scores, tricking an unshielded agent into making biased rejections.
  • Constructive Oracle Shield: Implemented a stateful fairness monitor that tracks Demographic Parity in real-time, actively intervening to override biased rejections into fair approvals for qualified candidates.
  • Reward Shaping Wrapper: Designed a custom Gymnasium wrapper that penalizes the Proximal Policy Optimization (PPO) agent when the shield intervenes, transferring the shield's "fairness knowledge" into the agent's core policy over time.
  • Comparative Auditing: Set up a testing pipeline to train, evaluate, and generate comparative plots analyzing the performance and fairness metrics of shielded versus unshielded agents.
