Safe-VerifiedAI

Reinforcement Learning Fairness Shield PPO Gymnasium

Overview

This project tackles a critical issue in AI: agents trained on biased historical data will naturally learn to discriminate against protected groups to maximize their baseline reward. To solve this, we implemented a Constructive Oracle Shield.

The framework monitors demographic parity in real-time, actively intervening on unfair decisions, and utilizes reward shaping to teach the agent to be fair over time.

The Pipeline

  • Synthetic Biased Environment: Built a custom CreditDataGenerator simulating a credit scoring environment where protected groups have artificially lowered scores, tricking an unshielded agent into making biased rejections.
  • Constructive Oracle Shield: Implemented a stateful fairness monitor that tracks Demographic Parity. It intervenes by overriding biased rejections into fair approvals for qualified candidates belonging to protected statuses.
  • Reward Shaping: Designed a custom Gymnasium wrapper that penalizes the Proximal Policy Optimization (PPO) agent when the shield intervenes. This transfers the shield's constraints into the agent's core policy.
